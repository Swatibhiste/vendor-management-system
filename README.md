# Vendor Management System

## Overview
This is a backend-focused project showcasing my skills in Java and Spring Boot.  
It demonstrates building a Vendor Management System with features like CRUD operations, API integration, and database management.  

## Tech Stack
- **Java**  
- **Spring Boot**  
- **RESTful APIs**  
- **MySQL**  
- **Git & GitHub**

## Key Features (Planned / Implemented)
- Vendor creation, search, and management  
- Skill management for vendors  
- Rate schedules and billing services  
- Basic validation and API error handling  
- Secure backend development practices

## Skills Highlight

- Database integration using MySQL  
- REST API design and integration
